Agent Harness Engineering

A short, hands-on learning path on how AI agents actually work under the hood — the engine, not the hype. Written for an engineer who learns by scanning, trying it, and coming back. Each topic is one sitting, ends with a small exercise you can run in Claude Code, and includes hand-picked videos if you prefer watching.

The Agentic LoopReady

The heartbeat of every agent: think → act → observe → repeat. What actually happens between your prompt and the answer, who decides when to stop, and what happens when a step fails.

Context EngineeringReady

The hard problem: deciding what goes into each agent’s limited memory window — compaction, summarization, and what a subagent should hand back to its parent.

SecurityReady
