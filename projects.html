VaultRAG: Full-Stack Local-to-Cloud RAG

Dual-mode RAG application demonstrating both traditional and multimodal RAG capabilities. Public mode leverages AWS Bedrock for cloud-based inference, while Private mode runs fully local with Ollama and Docling, ensuring no data ever leaves the machine.

  • Two distinct pipelines — text extraction with layout-aware document processing (PDF, DOCX, TXT) via Docling, and direct multimodal image embedding via Nomic Vision and Cohere Embed v4
  • Hybrid retrieval combining vector search, BM25, and metadata filtering with cross-modal retrieval enabling text queries against image collections in the same latent space

Agentic RAG - ReAct

A LlamaIndex ReAct agent that autonomously reasons about which tools to use for any given query. The LLM drives the decision loop — Reason, Act, Observe, Repeat — iterating through available tools until it has enough context to produce a final answer.

  • Built on LlamaIndex v0.14 AgentWorkflow with BedrockConverse for async streaming, enabling real-time token-by-token output as the agent reasons through its steps
  • Priority-based system prompt guiding the agent toward efficient retrieval — vector search first, file search, file read, with web search as fallback — without hard-coding routing logic

Agentic RAG - Orchestrated

A LangGraph state machine where the developer explicitly defines the execution flow. Unlike the ReAct approach where the LLM decides what to do next, here the graph controls routing between nodes. Each node calls the LLM for a specific, scoped task — classify, grade, rewrite, generate — with deterministic transitions.

  • Grader-Rewriter loop (max 2 retries) — after retrieval, grades document relevance and rewrites the query if documents are judged irrelevant, preventing infinite loops while improving retrieval quality
  • Hallucination-Regeneration loop (max 1 retry) — checks if the generated answer is faithful to retrieved documents, regenerating once if flagged as hallucinated before accepting the output
